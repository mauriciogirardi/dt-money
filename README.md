### :arrow_heading_down: How to install the project and run

```bash
  # clone the project
  git clone https://github.com/mauriciogirardi/dt-money.git
  # Open the project
  cd dt-money
  # Install
  yarn || npm i
  # run back-end fake
  npm run server || yarn server
  # run front-end
  npm run dev || yarn dev
  # open the application
   http://localhost:5173/
```
